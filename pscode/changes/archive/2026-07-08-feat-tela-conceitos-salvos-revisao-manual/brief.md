# Tela de conceitos salvos para revisão manual

## Objetivo
Uma tela para o usuário ver os conceitos que já ficaram salvos no glossário e
corrigi-los à mão — separada do fluxo de revisão dos pendentes.

## Comportamento esperado
- Item no menu lateral do app (janela "Aplicativo" do #95), depois de "Custos".
- Lista as correções salvas (errado → certo) e os termos mantidos ("é isso").
- Permite editar a grafia certa, remover uma entrada e adicionar uma correção do zero.

## Fora de escopo
- Mudar a lógica de autocorreção/detecção de conceitos em si.
- O painel de pendentes (badge → janela `#conceitos`) segue como está.
