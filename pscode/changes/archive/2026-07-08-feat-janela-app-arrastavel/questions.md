# Grill Me
- [x] Quais janelas ganham a barra arrastável? — Foco na janela unificada **"Aplicativo"** (#95); deve arrastar como um programa normal do PC. Conceitos fica fora deste card.
- [x] Como posicionar a barra na janela Aplicativo? — Faixa no topo (largura toda), com o botão fechar dentro dela.
- [x] Quais controles na barra? — Só fechar, reposicionado na barra (sem minimizar/maximizar; resize fora de escopo).
- [x] A barra exibe texto? — Sim, o **nome do app** ("PSTranscribe").
