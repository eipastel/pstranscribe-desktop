# Questions

- [x] Settings "nunca transparente"? → **Painel opaco, mantém cantos** (mantém `transparent:true`; só o fundo do `.settings-panel` vira opaco).
- [x] Vidro do widget 100% opaco — como tratar o frosted? → **Frosted opaco** (alpha do fundo → 1, mantém o `backdrop-blur` no CSS).
- [x] Slider de opacidade? → **Manter slider 50–100%** widget-only, default 100%.
