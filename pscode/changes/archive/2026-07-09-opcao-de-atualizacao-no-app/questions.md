# Perguntas

- [x] Onde a opção "Atualização" mora? → **Nova aba na sidebar**
  (Configurações / Custos / Conceitos / **Atualização**).
- [x] Como coexistir com o auto-update silencioso do startup? → **Manual controla
  o download**: `autoDownload = false`; no startup só checa e avisa, o download
  só acontece no clique "Baixar".
